Live Banking Market Dashboard

Interactive dashboard for major banks (Goldman Sachs, Morgan Stanley, J.P. Morgan, Citi, Bank of America, Barclays). Live prices, key metrics, technical indicators and short newsfeed.

Open live demo View source on GitHub

Quick notes

  • Data source: Yahoo Finance (yfinance).
  • News: NewsAPI (configure key in Streamlit secrets).
  • Built with Streamlit, Plotly and Pandas. Export CSV or short PDF summaries.
